Сайт для математиков седьмого класса.

Формула поддерживает операции {+,-,*,/}, умалчивать которые нельзя, а также функцию возведения в степень "pow(x,2)" (в данном случае x в квадрате). Имена переменных могут состоять из множества латинских букв и цифр, но на первом месте должна быть буква. Пример имени: VarName. Имя 2n - недопустимо. Поддерживаютса круглые скобки. Формат запросов вы можете легко понять по примерах.

Для формулы a*b+100
вычислить значения
при указанных a и b. 

	"a":[1,2,4,20]
	"b":[5,8,1,10]

Для формулы pi*2
вычислить значения
при указанных pi. 

"pi":[3.1,3.14,3.1415]

Для x в степени 0.5
вычислить значения
для множества x. 

"x":[100,25,9,4,1.327]

Упростить указанное выражение. 

-1-(c*2)-10+c*nn-3*nn+(c-nn)*2+35

Упростить указанное выражение. 

((a+b+100)/(b+a))*(11/a)

Упростить выражение: 

pow(a,2)*pow(a,3)+5

Вывести значение а из уравнения: 

5*a*a-7*a-16 = 0

Вывести неизвестные а и n из уравнения: 

a*a+2*a-300 = n+x+100*(n-x)

Выяснить равны ли формулы: 

n+1+x == (1+n*1)+(x-0)
